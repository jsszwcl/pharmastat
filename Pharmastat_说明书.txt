一、软件概述
------------
PharmaStat是一款面向药学研究人员的统计可视化桌面软件，支持组间比较、量效
曲线、相关性分析、热图和火山图等多种图表类型的生成与分析。

运行环境：Python 3.x，需安装 tkinter、matplotlib、seaborn、pandas、numpy、
scipy 等依赖库。

三、数据操作
------------
1. 手动输入：直接在表格中输入数据
2. CSV导入：通过 File > Import 导入CSV文件
3. CSV导出：通过 File > Save 保存数据

数据格式要求：
  • Group格式：CSV包含Group列和Sample1,Sample2...列
  • Dose-Response格式：CSV包含Dose,Effect,SD,N列
  • Correlation格式：CSV包含X,Y,Group列
  • Heatmap格式：无表头的数据矩阵
  • Volcano格式：CSV包含Gene,log2FC,P-value列


四、图表操作
------------
1. 保存为PNG：点击图表窗口的"Save PNG"按钮
2. 保存为PDF：点击图表窗口的"Save PDF"按钮
3. 打印：点击图表窗口的"Print"按钮


五、快捷键
----------
  Ctrl+S：保存项目


六、统计分析方法
----------------
  • 组间比较：独立样本t检验（与Control组对比）
  • 显著性标记：* p<0.05，** p<0.01，*** p<0.001
  • 量效曲线：Sigmoid函数拟合
    f(x) = bottom + (top-bottom)/(1+(EC50/x)^Hill)
  • 相关分析：线性回归（linregress）


七、软件界面布局
----------------
  窗口尺寸：1200x800 像素
  界面结构：
    菜单栏（File）
    └─ Notebooks（5个标签页）
       └─ 各标签页包含：
          · 输入区域（带滚动条）
          · 操作按钮栏
          · 指标/参数设置区


八、技术说明
------------
  图形库：matplotlib + seaborn
  统计库：scipy.stats, scipy.optimize
  数据处理：pandas, numpy
  GUI框架：tkinter

  字体设置：支持Microsoft YaHei、SimHei等中文字体
